## Using the window variable

One way to call a JavaScript function by string name, is using the global `window` object, which contains all the functions and variables defined in the current context, including the functions defined in the global scope.

```js

function helloWorld() {
    console.log('Hello World');
}

let functionName = "helloWorld"
window[functionName]();

```

## Calling from a module

If you're working with a JavaScript module and you want to call a function defined in that module using a string name, you need to export the function as part of the module's public interface. Here is an example:

```js

function helloWorld() {
    console.log('Hello World');
}

// Import the module and the function
import { helloWorld } from "./myModule.js";

// Call the function by its string name
const functionName = "helloWorld";
const moduleObject = { helloWorld };
moduleObject[functionName]();

```

## Using the exports object

If you only have the name of the function as a string and need to call it within a module, you can use the bracket notation to access the function as a property of the exports object, which is a reference to the public interface of the module. Here is an example:

```js

function helloWorld() {
    console.log('Hello World');
}

export function myOtherFunction() {
  // Call a function by name
  const functionName = "helloWorld";
  exports[functionName]();
}

```
